Ping

This item only works when running on the server. ServerThis item is only intended to be used by the module's authors. Private

Interfaces with MemoryStoreService to persist event timing across servers. Uses a MemoryStoreHashMap keyed by event name, with values being the unix endTime timestamp.

On require, this module:

  1. Defers Boot to initialize the system
  2. Defers Polling to begin the background sync loop

The hash map entries expire after admin_event_duration + 60 seconds as a safety margin.

The calculated unix end-time timestamp

Writes a new end-time for the given event key to MemoryStoreService and returns it.

The end-time is calculated as os.time() + admin_event_duration. The hash map entry is set to expire after admin_event_duration + 60 seconds.
