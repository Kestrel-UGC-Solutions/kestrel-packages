State

This item only works when running on the server. ServerThis item is only intended to be used by the module's authors. Private

A lightweight observable value container. Holds a single value and notifies registered callbacks whenever that value changes. Used internally by AdminAbuse for reactive state management. 

local myState = State.new(0)

myState:Observe(function(value)
	print("Value changed to:", value)
end)

myState:Set(42) -- prints "Value changed to: 42"

Updates the stored value. If the new value is different from the current value, all registered observer callbacks are invoked with the new value via task.spawn.

If the new value is the same as the current value, this is a no-op.

Unsubscribe function

Registers a callback that fires whenever the value changes. Optionally invokes the callback immediately with the current value.

Returns an unsubscribe function — call it to stop receiving updates.

local unsubscribe = myState:Observe(function(value)
	print(value)
end, true) -- true = also fire immediately with current value

-- Later, to stop observing:
unsubscribe()
