AdminAbuse

This item only works when running on the server. Server

The main entry point for the Admin Abuse package. Provides a simple API to trigger timed events that automatically synchronize across all active game servers.

Install via Wally:

[dependencies]
AdminAbuse = "kestrel/admin-abuse@0.1.0"

Basic usage:

local AdminAbuse = require(path.to.AdminAbuse)

-- Trigger an event on every server
AdminAbuse.event("SixSeven")

-- React to event changes
AdminAbuse.onEventChanged:Connect(function(eventKey)
	print("Event activated:", eventKey)
end)

On require, the module subscribes to MessagingService to receive events broadcast by other servers. The subscription retries up to 2 times with a 5-second delay between attempts if it fails.

AdminAbuse.onEventChanged: Signal

A Signal that fires whenever a new admin event begins executing. The callback receives the event key as its only argument.

AdminAbuse.event(
eventKey: string--

The key identifying which event to trigger

) → ()

Triggers an admin abuse event across all servers. The event is identified by its key from the Events module.

When called, this function:

  1. Validates the event key and its Executor
  2. Persists the event end-time in MemoryStoreService via the Ping module
  3. Broadcasts the event to all servers via MessagingService
  4. Activates the event locally (re-triggers if the same event is already active)
AdminAbuse.event("SixSeven")
